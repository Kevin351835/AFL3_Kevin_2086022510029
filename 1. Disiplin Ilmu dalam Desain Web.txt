1. Disiplin Ilmu dalam Desain Web
Desain web bukan hanya sekadar satu disiplin ilmu, melainkan gabungan dari beberapa pengetahuan:


Graphic Design: Menentukan tampilan visual secara keseluruhan seperti tulisan, warna, gambar, dan tata letak (layout).


Information Design: Menentukan cara mengorganisasi konten web dan metode pengaksesannya, biasanya menggunakan flow chart dan diagram.


Interface Design: Berfokus pada metode interaksi pengguna dan navigasi konten di dalam web.


Document Production: Pembuatan dokumen web, style sheet, skrip, dan gambar.


Scripting & Programming: Bertanggung jawab atas konten interaktif (seperti formulir dan konten dinamis) serta penguasaan basis data.


Multimedia: Mengelola konten audio, video, dan animasi.

2. Konsep Usability (Keterpakaian)
Dalam merancang antarmuka (interface), terdapat konsep usability yang meliputi:


Learnability: Seberapa mudah pengguna menyelesaikan tugas dasar pada pertemuan pertama.


Efficiency: Seberapa cepat pengguna melakukan tugas setelah mempelajari desain tersebut.


Memorability: Seberapa mudah pengguna kembali mahir setelah tidak menggunakan desain dalam waktu lama.


Errors: Jumlah dan tingkat keparahan kesalahan yang dilakukan pengguna.


Satisfaction: Tingkat kenyamanan pengguna saat menggunakan desain.

3. Teknologi Utama Web

HTML/XHTML: Bahasa penerjemah untuk mengidentifikasi komponen dokumen (paragraf, daftar, tabel, dll.).


CSS (Cascading Style Sheet): Standar untuk memformat teks, elemen web, dan tata letak halaman agar lebih rapi dan konsisten antar browser.


Web Design Layers: Terdiri dari Structure Layer (HTML), Presentation Layer (CSS), dan Behaviour Layer (interaktivitas).

4. Tren Desain Web

Responsive Design: Memastikan tampilan situs berubah secara visual sesuai perangkat atau ukuran layar tanpa kehilangan fungsi atau konten.


Flat Design: Tren desain minimalis yang populer (terinspirasi dari UI Windows 8).


Infographics: Penggunaan gambar visual seperti bagan atau diagram untuk menyampaikan data secara menarik.

5. Strategi Profesional Web Designer
Untuk menjadi desainer web profesional yang baik, diperlukan tiga pelajaran penting:


Tetap Up-to-date: Selalu mengikuti perkembangan teknologi (seperti HTML5 dan CSS3) dan tren desain terbaru.


Differentiate Yourself: Memiliki Unique Selling Proposition (USP) atau nilai unik yang membedakan Anda dari desainer lain.


Under-promise and Over-deliver: Memberikan hasil yang melebihi ekspektasi klien untuk membangun kepercayaan dan reputasi.
